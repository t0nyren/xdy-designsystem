同一只熊猫
走进四个片场

把 PANDA YEAH 2.0 放进四种动画语言里做定妆测试——统一场景（清晨饺子铺·捏褶·蒸笼热气）便于横向对比；识别资产在任何风格里不动：饺子褶耳朵、黑白配色、红 T 恤。定妆之后，用表现最好的风格出分镜与视频样片。

00 · Character Bible 角色圣经

PANDA YEAH · Dumpling MasterAAA 制作级角色手册：主插画 + 三视图 + 同伴角色「饺子精皮皮」三视图 + 细节/配件（蒸笼·擀面杖·围裙·红绳铃）/ 色板。动漫化的一切风格演绎以此为准。

01 · Style Frames 四风格定妆

Style A · 基准

手绘暖卡通（现行 2.0）

暖墨勾线 + 平涂 + 纸纹。品牌一致性最强，周边与平面天然通用。

Style B · 日漫

吉卜力水彩

水彩背景 + 赛璐璐角色 + 光尘。烟火气与「现包的手作感」最契合，适合品牌短片。

Style C · 3D

迪士尼 / 皮克斯 3D

大眼神 + 绒毛质感 + 电影级光。表情张力最大，适合节日 campaign 与广告位。

Style D · 国风

水墨动画

宣纸留白 + 墨韵热气 + 一枚红印。接「包好中国饺子·传承千年文化」叙事，适合文化向内容。

02 · Storyboard 短片《清晨的一屉》四格

故事：天没亮的饺子铺，熊猫师傅拌好当日的馅（起）→ 案板前捏出月牙褶（承）→ 蒸笼揭盖、热气涌起（转）→ 把第一屉端给等在门口的小客人，比耶（合）。

Storyboard · 吉卜力水彩版四格一张出（storyboard 多格能力），角色与场景全程一致；每格即视频样片的首帧。
Storyboard · 迪士尼 3D 版同一故事换 3D 语言，对比叙事表现力。

04 · Usage 风格分工建议

手绘卡通

周边、包装、门店贴纸、社媒日常——品牌默认。

吉卜力水彩

品牌短片、节气/上新短内容——讲手作与烟火气。

迪士尼 3D

春节等大 campaign、商场大屏、期间限定 TVC。

水墨

文化副线「包好中国饺子」专用，博物馆/外交场合物料。
