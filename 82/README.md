# 八二小区抄手独立官网体系

本目录是八二小区抄手的独立官网项目，不属于熊大爷 `/web/` 网站。

## 目录边界

- `index.html`：八二项目与文件中心首页。
- `site/`：八二官网六页响应式预览。
- `assets/`：八二品牌与产品网页素材。
- `docs/`：八二策划、信息架构与视觉规范。

## 部署边界

- 独立预览域名：`82.secondlife.today`。
- 独立正式域名、容器、后台配置、数据、日志、上传和备份由后续正式部署确定。
- 不引用熊大爷 `content.json`、后台模板、`/web/` 资源或业务数据。
- 两个品牌只可共用服务器、Docker 等基础设施层。
