Design System · IP 2.0 & Merchandise

PANDA YEAH 2.0 · 周边设定

憨要憨得高级
周边要背得出门

IP 重设计只换气质、不换识别：饺子褶耳朵、比耶手势、红 T 恤一个不动，把廉价 3D 公仔换成手绘暖线条的 2D 角色——能上周边、能做动效、能进插画。周边按「日常被看见的频率 × 与饺子场景的贴合度」选了六类。

01 · Character 角色设定

三视图 Turnaround2.5 头身；纯黑耳/眼斑/四肢，白头身，正红 T 恤；耳朵为月牙饺褶形。暖墨勾线 + 平涂 + 纸纹底。
表情库 Expressions八格常用表情：笑/大笑/惊讶/困/心动/大厨脸/comic 哭/眨眼，社媒与包装贴纸直接取用。
动作库 Poses六个品牌动作：捏褶/举蒸笼/拎帆布袋/吃饺子/比耶/鞠躬迎客——覆盖门店、外卖、周边、动效四类场景。
Keep 不许动

饺子褶耳朵、比耶手势、红 T 恤、黑白熊猫配色。这四样是识别资产，任何应用不得改。

Style 画法

暖墨勾线 + 低饱和平涂 + 纸纹面粉白底；禁 3D 渲染、禁发光描边、禁塑料高光。

Usage 用法

IP 与摄影不同屏混排；深底用白版剪影；最小可用尺寸 24px（只保留头部）。

02 · Merchandise 周边六类

选类逻辑：帆布袋与围裙直接长在「生包外带」和「现包」两个品牌场景上；杯帽是日常曝光位；徽章冰箱贴用 8 款饺子拟人做低客单集邮款；饭盒把「吃饺子」带进办公室。所有周边共用一套语言：面粉白画布、暖墨线、绛红只做一枚印、金只做细节。

帆布袋本色帆布；正面 IP 拎蒸笼，背面褶弧一排 + 绛红小印。
围裙门店与周边同款；胸口绛红小印，下摆淡金褶弧，口袋 IP 绣标。
马克杯 + 保温杯面粉白杯身 IP 小图 + 口沿金细线；保温杯绛红底白 IP。
渔夫帽 + 棒球帽面粉白渔夫帽绣 IP 头 + 侧面小饺子；绛红棒球帽白绣。
徽章 / 冰箱贴套装8 款饺子拟人 + 1 枚 IP 头像，集邮款；对应 8 款常青饺子。
保温饭盒 + 筷子套装双层面粉白盒 + 原木色盖带；盖面 IP 举蒸笼 + 绛红小印。

03 · Merchandise 周边第二批

第二批补齐穿戴、收藏与日用三条线：T恤卫衣是行走的主视觉；毛绒公仔和饺子抱枕是情感款；钥匙扣、贴纸是低客单高频款（贴纸直接取自表情库）；长柄伞把绛红条纹带上街；文具套装进办公室。

T恤 + 卫衣面粉白 T 恤胸位比耶 IP；燕麦色卫衣左胸绣标、后领一排小饺子。
毛绒公仔 + 饺子抱枕IP 本体坐姿公仔（刺绣脸）；月牙饺抱枕带睡颜与褶线车缝。
钥匙扣挂件软胶 IP 全身 + 迷你饺子挂坠 + 织带短挂绳（绛红小印皮标）。
贴纸表情包套装八枚模切贴纸直接取自表情库 + 两枚小饺子；社媒与包装同源。
长柄伞面粉白×绛红相间伞面，单幅 IP 小图 + 单幅饺子母题，原木弯把。
笔记本文具套装面粉白硬壳本（右下压凹 IP + 绛红束带）+ 饺子形便签 + 原木铅笔。
Print 印法

帆布类丝印 2 色（暖墨 + 绛红）；刺绣类 IP 只用头部简化版；陶瓷用釉上彩 + 金线手工描边档位可选。

Color 用色

周边底色只允许：本色帆布 / 面粉白 / 绛红 / 暖墨四选一。禁渐变、禁荧光、禁大面积亮红（亮红只活在 LOGO）。
